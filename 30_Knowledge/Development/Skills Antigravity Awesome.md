## 📦 Installation & Updates

### Sync Skills Repository
```bash
# Clone repository
git clone https://github.com/sickn33/antigravity-awesome-skills.git ~/.agent/skills/antigravity-awesome-skills

# Update skills
cd ~/.agent/skills/antigravity-awesome-skills
git pull origin main
```

### Create Symlinks for All IDEs
```bash
# For Cursor
ln -sf ~/.agent/skills/antigravity-awesome-skills ~/.cursor/skills/

# For Antigravity
ln -sf ~/.agent/skills/antigravity-awesome-skills ~/.gemini/antigravity/skills/

# Verify symlinks
ls -la ~/.cursor/skills/
ls -la ~/.gemini/antigravity/skills/
```
