# RAPPORT — REBRANDING recall-agency.com

## 1. ANALYSE DE TITLEMAN.AI (Source)

### Design System
| Élément | Détail |
|---------|--------|
| **Framework** | Nuxt.js (Vue.js) |
| **Styling** | CSS custom properties, pas Tailwind |
| **Fonts** | Inter (sans), Merriweather (serif), JetBrains Mono (code) |
| **Palette** | Primary Blue `#4561eb`, Background Warm off-white `#faf9f5`, Dark mode noir profond `#0c0b09` |
| **Esthétique** | SaaS professionnel haut de gamme, sections full-width, cartes 560x560px, espacement généreux (40px padding, sections 80-120px gap) |
| **Animations** | Logo marquee scroll infini, hover states lisses, boutons avec border-radius 6px |

### Structure de Page
```
Header (84px) → Hero (630px) "AI for Real Estate" → Intro value prop →
Social proof bar → Logo marquee (30+ firms) → Products (3 cards) →
Stats (2560 sources, 140 DBs, 190 pays) → Solutions (tabs 4 verticals) →
Enterprise features (4 colonnes) → Footer CTA → Footer
```

### Points Clés du Contenu
- **CTAs principaux**: "Get access" (3x), "Explore" (7x links)
- **Social proof**: Logos Fidelity, Blackstone, J.P. Morgan, CBRE, etc.
- **Positionnement**: "Professional-class AI for real estate firms"

| Fichier | Contenu |

### Brand Voice Summary
- **Ton**: Sharp, Precise, Controlled — "strategic over tactical, direct over polite, elite over mass"
- **Key Pillars**: Control, Leverage, Security, Real+Digital Fusion, Intelligence Layer
- **Vocabulary**: systems, architecture, leverage, intelligence, control, infrastructure, signal, edge, asymmetry, scale
- **Avoid**: passion, journey, innovative, cutting-edge, customer-centric
