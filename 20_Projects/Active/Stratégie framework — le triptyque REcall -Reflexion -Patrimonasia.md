⚙️ 2️⃣ Méthode API interne (microservice Crawl4AI)

🧩 Principe: tu transformes ton script en microservice FastAPI qui tourne en tâche de fond.

n8n appelle ton endpoint via un HTTP Request node.

Exemple rapide:

Copy code

Python

\# /opt/crawl4ai/app.py

from fastapi import FastAPI, Query

from crawl4ai import Crawl4AI

from crawl4ai.llm import OllamaLLM

  

app = FastAPI()

llm = OllamaLLM(model="phi3", base\_url="http://localhost:11434")

crawler = Crawl4AI(llm=llm)

  

@app.get("/scrape")

def scrape(url: str = Query(...)):

result = crawler.scrape(url, mode="markdown")

return result
