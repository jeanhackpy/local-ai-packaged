# Zero-shot Sequential Neuro-symbolic Reasoning for Automatically Generating Architecture Schematic Designs

## Abstract

This paper introduces an automated system for generating architecture schematic designs, leveraging the combined strengths of generative AI (neuro reasoning) and mathematical program solvers (symbolic reasoning). The sequential neuro-symbolic reasoning approach emulates traditional architecture design processes from initial concept to detailed layout.

## Key Innovation

### Sequential Neuro-Symbolic Reasoning
- **Neuro reasoning**: Generates constraints and cost functions
- **Symbolic reasoning**: Uses mathematical solvers to solve optimization problems
- **Feedback loops**: Tight integration between both reasoning types at each design stage

### No Hand-crafted Cost Functions
Unlike traditional approaches, this method uses GPT-4 (without further training) to:
1. Generate constraints from design requirements
2. Generate cost functions that symbolic solvers can optimize
3. Iterate through design stages automatically

## Applications

- Multifamily real estate development
- Building schematic design automation
- Neighborhood-aware design generation
