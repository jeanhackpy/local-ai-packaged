### Inspection du site

Vous pouvez inspecter les requêtes réseau effectuées par le site pour voir si elles utilisent une API interne. Voici comment faire :

#### a. Ouvrir les outils de développement
- **Google Chrome** : Cliquez sur les trois points verticaux dans le coin supérieur droit > Plus d'outils > Outils de développement.
- **Firefox** : Cliquez sur les trois lignes horizontales dans le coin supérieur droit > Développeur web > Outils de développement.

#### b. Aller à l'onglet "Network"
- Visitez FazWaz.com et naviguez sur le site tout en surveillant les requêtes réseau dans l'onglet "Network".

#### c. Rechercher des requêtes API
- Cherchez des requêtes vers des URL qui ressemblent à des endpoints d'API, souvent elles contiennent `/api/` dans l'URL ou ont des formats JSON dans les réponses.
