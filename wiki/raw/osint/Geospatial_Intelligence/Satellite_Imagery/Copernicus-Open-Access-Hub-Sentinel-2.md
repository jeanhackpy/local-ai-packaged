### Copernicus Open Access Hub (Sentinel-2)
- **Coût:** Gratuit
- **Accès:** Les images Sentinel-2 sont disponibles gratuitement via le Copernicus Open Access Hub. Vous devez créer un compte gratuit pour accéder aux données.
- **API:** Copernicus propose une API pour rechercher et télécharger les données Sentinel. Pour plus d'informations, consultez la [documentation de l'API Copernicus](https://scihub.copernicus.eu/userguide/BatchScripting).
