### 9. Full Example for ARM
Here’s a complete example to run an ARM system with a disk image using the `virt` machine type:

```bash
qemu-system-aarch64 \
  -machine virt \
  -cpu cortex-a53 \
  -m 1024 \
  -drive file=disk_image.img,if=none,id=drive0 \
  -device virtio-blk-device,drive=drive0 \
  -nographic
```
