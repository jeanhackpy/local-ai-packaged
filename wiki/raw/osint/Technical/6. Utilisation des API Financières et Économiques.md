### 6. **Utilisation des API Financières et Économiques**
Exemple d'utilisation de l'API Alpha Vantage pour obtenir des données financières :
```python
from alpha_vantage.timeseries import TimeSeries

ts = TimeSeries(key='YOUR_API_KEY', output_format='pandas')
data, meta_data = ts.get_daily(symbol='AAPL')
print(data.head())
```
