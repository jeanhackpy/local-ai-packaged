### Hardware Components:
1. **Raspberry Pi 4 (4GB or 8GB RAM):**
   - The main computer that will run the application.
2. **Coral USB Accelerator:**
   - Provides the ML acceleration needed for real-time inference.
3. **Camera Module:**
   - Raspberry Pi Camera Module or any compatible USB camera for capturing video feed.
4. **MicroSD Card:**
   - At least 32GB for the OS and application storage.
5. **Power Supply:**
   - A reliable power supply for the Raspberry Pi.
6. **Optional: Sensors:**
   - Additional sensors like motion detectors for enhanced functionality.
