### Software Stack:
1. **Operating System:**
   - Raspberry Pi OS (formerly Raspbian) or a lightweight Linux distribution.
2. **TensorFlow Lite:**
   - For running machine learning models.
3. **OpenCV:**
   - For image processing and computer vision tasks.
4. **Flask/Django:**
   - For creating the web interface for remote monitoring.
5. **MQTT or Websockets:**
   - For real-time communication and alerts.
6. **SQLite/PostgreSQL:**
   - For data storage and logging.
