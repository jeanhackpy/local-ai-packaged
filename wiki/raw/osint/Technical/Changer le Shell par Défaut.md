### Changer le Shell par Défaut

Si vous souhaitez changer votre shell par défaut entre `zsh` et `bash`, vous pouvez le faire en utilisant la commande `chsh` (change shell) :

1. **Changer vers `bash` :**
   ```bash
   chsh -s /bin/bash
   ```

2. **Changer vers `zsh` :**
   ```bash
   chsh -s /bin/zsh
   ```

Vous devrez peut-être redémarrer votre terminal ou ouvrir une nouvelle session pour que le changement prenne effet.
