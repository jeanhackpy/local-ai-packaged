### AWS Amplify

#### Caractéristiques
- **Déploiement et Hébergement**: Déploiement automatique à partir des dépôts Git, hébergement rapide sur un CDN global.
- **Fonctionnalités Back-end**: Intégration avec les services AWS pour des fonctionnalités comme l'authentification, les APIs GraphQL, et le stockage.
- **Écosystème AWS**: Accès à une large gamme de services AWS pour étendre les capacités des applications.
- **Interface Utilisateur**: Console de gestion complète et intégrée avec AWS.

#### Cas d'Utilisation Idéal
- Développeurs recherchant une intégration étroite avec les services AWS et la capacité d'ajouter facilement des fonctionnalités back-end à leurs applications Jamstack.
