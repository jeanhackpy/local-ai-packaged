### Fleek

#### Caractéristiques
- **Intégration IPFS**: Fleek utilise IPFS pour l’hébergement décentralisé de sites Web et d’applications.
- **CDN Décentralisé**: Combinaison de IPFS avec un CDN pour améliorer les performances.
- **Déploiement Continu**: Intégration avec Git pour des déploiements automatiques.
- **Compatibilité Web3**: Support natif pour les domaines ENS (Ethereum Name Service) et d’autres technologies Web3.

#### Cas d'Utilisation Idéal
- Développeurs Web3 recherchant une solution simple pour déployer des applications décentralisées avec IPFS.
