## Computer Vision

Application de deep learning à l'analyse d'images/vidéos.

### Architectures principales

- **CNN** : ConvNets (ResNet, EfficientNet, ConvNets patch-level)
- **ViT** : Vision Transformers (self-supervised learning)
- **GCN** : Graph Neural Networks (spatial data, floor plans)
- **Mask-RCNN** : Instance segmentation
- **Siamese networks** : Similarity learning

### Applications

- Property image analysis
- Building condition assessment
- Satellite/aerial imagery
- Floor plan parsing
