## Résumé

Deep learning multi-modal pour prédiction de prix immobilier (Hasan et al. 2024). Embedding joint de attributs bruts + texte + images + données géo-spatiales. Surpasse significativement les modèles single-modality.

## Points clés

- Architecture multi-modale jointe : attributes + text + images + geo-spatial
- Surperformance vs modèles single-modality sur benchmarks
- Capture d'informations complémentaires entre modalités
- Application à la valorisation immobilière thaïlandaise
