# Solana Token Analysis Model

## Summary

Python script template for extracting and analyzing Solana tokens using CoinGecko API. Three-step framework: Data Extraction (fetch_solana_tokens via requests), Data Analysis (identify top 10 performers by 24h price change), Pattern Detection (correlation matrix, linear regression for market cap vs price change, matplotlib scatter plots). Uses pandas, sklearn, matplotlib. Outputs results to CSV and console.

## Key Concepts

- CoinGecko API for crypto market data
- Pandas DataFrame for token analysis
- sklearn LinearRegression for pattern detection
- Market cap correlation with price movement
- matplotlib visualization of relationships
- CSV export for further analysis
