# Analyze Threat Report - Cybersecurity Report Extraction

## Summary

A super-intelligent cybersecurity expert framework for extracting surprising, insightful information from threat reports. Produces one-sentence summary, up to 50 trends, 15-30 statistics, 15-30 quotes, references (tools/companies/projects), and 15-30 recommendations.

## Key Concepts

- **One-Sentence-Summary**: 25-word plain language capture, no jargon or marketing
- **Trends**: Minimum 20, up to 50 surprising/interesting trends from report
- **Statistics**: 15-30 surprising statistics with exact values
- **Quotes**: 15-30 exact quotes from the input
- **References**: All mentioned tools, applications, companies, projects
- **Recommendations**: 15-30 actionable insights

## Connections

- Used in vulnerability_management workflows
- Applied in threat_modeling for security assessment
