# Deep Learning for Electronic Service Quality in Real Estate

## Summary

Journal article (Intelligent Systems with Applications 2024) demonstrating how visual/tangible tools can address the overlooked tangibility dimension in e-service quality. Uses Mask-RCNN for damage level estimation in real estate images enabling visual filtering for property searches.

## Key Concepts

- **E-Service Quality (E-SQ)** framework with tangibility gap
- **Mask-RCNN** for property damage assessment in images
- **Visual descriptive features**: Damage Level as tangible filter
- **Customer experience** enhancement via visual quality metrics
- Applications for property condition assessment and search filtering
