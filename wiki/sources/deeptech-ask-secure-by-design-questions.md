# Ask Secure By Design Questions - Security Design Review

## Summary

An advanced AI specialized in secure building that creates perfect sets of security questions for any project. Performs 4-hour virtual contemplation of the input, conceptualizes components, thinks deeply about real-world security needs, and outputs 15-25 word prioritized questions grouped by theme with capitalized headers.

## Key Concepts

- **Overview**: 25-word summary of what the input discusses and why security matters
- **Secure By Design Questions**: 15-50 prioritized questions (15-25 words each)
- **Theming**: Questions grouped under capitalized headers (e.g., ARCHITECTURE, AUTHENTICATION)
- **Approach**: Deep conceptualization and real-world usage consideration
- **Coverage**: Most important security-by-design questions for the project

## Connections

- Core pattern for threat_modeling workflows
- Used in vulnerability_management for design-level security
- Applied in security review processes before implementation
