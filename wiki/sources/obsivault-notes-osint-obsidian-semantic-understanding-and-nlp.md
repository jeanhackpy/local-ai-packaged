# OSINT - Obsidian, Semantic Understanding, and NLP

## Summary

Detailed tutorial on using Obsidian as an OSINT analysis powerhouse through semantic understanding and NLP. Introduces concept of "Intelligence Playground Vault" using Wikidata imports to create semantic triples (subject-predicate-object). Key plugins: Wikidata Importer, Wikipedia, Graph Link Types, Juggl, SkepticMystic's NLP and Graph Analysis. Use cases: entity relationships, topic modeling (HITS, Jaccard algorithms), network visualization, semantic canvas for report generation. Demonstrates how Obsidian's linked notes create inferred relationships beyond explicit links.

## Key Concepts

- Wikidata semantic enrichment for OSINT vaults
- Triples: subject-predicate-object structured statements
- Transitive reasoning: A→B, B→C = A→C inferred
- HITS algorithm: hub vs authority scoring
- Jaccard algorithm: similarity between notes
- Juggl plugin for interactive graph visualization
- Semantic canvas for investigative reporting
