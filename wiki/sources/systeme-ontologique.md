## Résumé

Système ontologique pour les données immobilières thaïlandaises : hiérarchie de localisation (province → district → sous-district → BTS/MRT) vers types de propriétés (condo, maison, terrain) vers équipements vers couches de prix avec règles d'inférence. Fondation pour données immobilières structurées.

## Points clés

- Hiérarchie de localisation : province > district > sous-district > station BTS/MRT
- Taxonomie des types : condo, maison individuelle, terrain, mixed-use
- Modèle de caractéristiques : équipements, état, âge, étage
- Couches de prix avec règles d'inférence géographique
- Base pour système de recommandation et évaluation automatisée
