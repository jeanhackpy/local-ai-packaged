## Project Blueprint: REcall OS

Master document defining the vision, architecture, and execution plan for REcall OS — "Palantir for Real Estate" in Southeast Asia.

### Core Vision

Build the definitive intelligent infrastructure for Thai real estate market — not another listings portal, but an AI-powered operating system.

### Three Brands

| Brand | Purpose | Target |
|-------|---------|--------|
| **REcall Agency** | Technology & cybersecurity core | B2B consultancy |
| **Reflexion.asia** | Premium marketplace | B2C/B2B Thailand |
| **Patrimonasia.com** | EU investor bridge | B2B European market |

### Technical Stack

- **Frontend**: Next.js + TailwindCSS + shadcn/ui
- **Data**: Supabase (SQL), Qdrant (vectors), Neo4j (graphs), Redis
- **AI**: OpenRouter + Ollama (local)
- **Orchestration**: n8n + LangGraph
- **Pipeline**: Crawl4AI + Pydantic
- **3D**: Unreal Engine 5 + Cesium

### 21-Day MVP Plan

1. **Days 1-3**: Landing page + lead capture
2. **Days 4-10**: Crawl4AI → 200-800 listings
3. **Days 11-21**: Onboard 20 agents + "man-behind-curtain" AI

### Key Features

- Intelligent Onboarding Quiz (swipe-based)
- AI Real Estate Expert Agent (RAG on multi-DB)
- Agent Matching & Co-Brokerage Engine
- Reputation Shield (cybersecurity for agencies)
- Immersive 3D/AR/VR Layer
- Web3 Integration (blockchain logging, DID auth)
