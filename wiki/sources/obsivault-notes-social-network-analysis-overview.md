# Social Network Analysis Overview

## Summary

Article on social network analysis (SNA) from theory to Python applications. Covers network basics: nodes (entities) and edges (connections) with properties. Real-world network characteristics: Small World phenomenon (six handshakes), Scale-Free networks (power-law distribution, hub nodes), Homophily (similarity clustering). Centrality measures: Degree, EigenVector/PageRank, Closeness, Betweenness. Information flow models: Linear Threshold (accumulated influence) and Independent Cascade (probabilistic spread). Influence maximization as NP-Hard problem. Game of Thrones network case study demonstrates diffusion dynamics.

## Key Concepts

- Nodes and edges as network fundamentals
- Small World: short paths between network members
- Scale-Free: hub nodes with power-law distribution
- Centrality measures for identifying key nodes
- Independent Cascade: viral spread model
- Linear Threshold: accumulated influence threshold
- Influence maximization: selecting seeding nodes
- Game of Thrones network for diffusion simulation
