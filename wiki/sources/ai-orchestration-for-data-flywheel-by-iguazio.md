# AI Orchestration for Data Flywheel

## Summary

NVIDIA blueprint combining Iguazio MLRun's orchestration engine with NVIDIA NeMo microservices for production-integrated data flywheels. Automates end-to-end AI agent optimization with continuous retraining and evaluation.

## Key Concepts

- **MLRun** low-code orchestration by Iguazio (QuantumBlack/McKinsey)
- **NeMo Customizer** for model fine-tuning (SFT-LoRA)
- **NeMo Evaluator** for automated model and workflow evaluation
- **Continuous optimization** reducing latency and TCO
- **NIM models**: Llama 3.3 70B, 3.2 1B/3B, 3.1 70B/8B
