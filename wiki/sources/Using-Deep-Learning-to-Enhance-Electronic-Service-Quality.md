# Using deep learning to enhance electronic service quality: Application to real estate websites

## Abstract

Electronic service quality (E-SQ) is a strategic metric for successful services. While most e-service quality dimensions are well-studied, **tangibility** is overlooked. This research demonstrates how visual/tangible tools can balance the intangible nature of e-services using deep learning for computer vision.

## Key Innovation

### Novel Visual Descriptive Feature: Damage Level
- Uses **Mask-RCNN** to estimate damage in real estate images
- Enables tangible visual filtering for property searches
- Enhances tangibility and efficiency of e-services

## Problem Addressed

Users typically have specific search criteria but most services lack flexible visual filters, such as:
- Damage Level to property
- Visual condition assessment
- Property quality metrics

## Solution Approach

1. **Visual feature extraction** using Mask-RCNN
2. **Damage Level classification** as a tangible feature
3. **Integration** into e-real estate services
4. **Enhanced customer experience** through visual filtering
