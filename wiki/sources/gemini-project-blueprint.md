# REcall Project Blueprint

## Summary

Master strategic document for REcall OS - an AI-powered "Palantir for Real Estate" platform targeting Southeast Asia starting with Thailand. Version 1.0 dated 2024-12-14. The blueprint covers the complete vision, brand architecture (REcall Agency, Reflexion.asia, Patrimonasia.com), technical stack, go-to-market strategy for a solo founder with zero capital, and a 21-day MVP execution plan.

## Key Concepts

- **REcall OS Vision**: Sovereign, AI-powered operating system unifying real-time data, expert AI agents, and immersive 3D environments
- **Three Brands**: REcall Agency (tech consultancy), Reflexion.asia (B2C/B2B marketplace), Patrimonasia.com (B2B European investor channel)
- **Sovereign Tech Stack**: Next.js, Supabase (PostgreSQL), Qdrant (vector), Neo4j (graph), Redis, n8n, LangGraph/Flowise, Ollama, Crawl4AI, Pydantic
- **3D/Immersive Layer**: Unreal Engine 5 + Cesium for digital twins and virtual property tours
- **21-Day MVP**: Landing page (days 1-3), scraping pipeline 200-800 listings (days 4-10), agent network + "illusion of AI" (days 11-21)
- **Target**: PropTech Venture Studio partnership for team/funding while retaining product leadership
- **Key Features**: RAG-based AI agent, Agent Matching Graph, Reputation Shield (cybersecurity), UE5/Cesium 3D layer, Web3 integration
