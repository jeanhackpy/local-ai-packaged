## Résumé

NVIDIA blueprint for ambient AI voice agents in healthcare. Dual-agent: Provider Agent (clinical SOAP documentation) + Patient Agent (intake/scheduling). Parakeet ASR + NeMo Guardrails + Llama Nemotron. HIPAA-aligned.

## Points clés

- Dual-agent architecture: Provider Agent (clinical documentation) + Patient Agent (intake/scheduling)
- Parakeet ASR for speech recognition, Llama Nemotron for LLM reasoning, NeMo Guardrails for safety
- HIPAA-aligned workflow for clinical documentation and patient intake
