## Résumé

La confiance comme problème d'infrastructure de données. Framework confiance ↔ infrastructure ↔ données comme avantage concurrentiel. Data provenance et lineage comme signaux de confiance pour les plateformes immobilières.

## Points clés

- Trust comme moat compétitif via accumulation de données vérifiables
- Infrastructure de données crée confiance entre buyers/sellers
- Data provenance et lineage pour transparence des évaluations
- Application au marché immobilier thaï : trust deficit comblé par données
