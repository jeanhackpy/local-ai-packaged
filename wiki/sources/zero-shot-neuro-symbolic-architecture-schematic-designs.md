# Zero-shot Sequential Neuro-Symbolic Architecture Design

## Summary

ArXiv paper (2402.00052) introducing automated system for generating architecture schematic designs using GPT-4 (neuro reasoning) + mathematical solvers (symbolic reasoning). Sequential approach emulating traditional design processes from concept to detailed layout.

## Key Concepts

- **Neuro reasoning**: GPT-4 generates constraints and cost functions (zero-shot, no training)
- **Symbolic reasoning**: Mathematical solvers optimize design problems
- **Feedback loops**: Tight integration between reasoning types at each stage
- **No hand-crafted cost functions**: Learns from design requirements directly
- Applications: multifamily development, schematic automation, neighborhood-aware design
