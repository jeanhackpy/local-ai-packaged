# Techno Mix Bangkok Underground Adaptation - Grok

## Summary

Comprehensive techno set documentation for Bangkok underground clubs with multiple variations. Includes standard techno BKK version (3h30-4h) with six phases, warehouse dark version (3h45) with industrial loops, and Red Axes variant integration. BPM ranges from 122-128 across phases. Key tracks: Garnier (1-4 Doctor C'est Chouette, Breathless, Crispy Bacon, Flashback), Hood (Minus, Chase, Power To Prophet), Mills (The Bells, The Occurrence, Gamma Player), PAS (Arc, Kat, Booster). Booth rules emphasize: BPM +1 for warehouse, dirty bass, loops only, zero talk.

## Key Concepts

- Three set variations: Techno BKK, Warehouse Dark, Red Axes Variant
- Industrial build phase with continuous loops, no cuts
- Abyssal pressure maintained without emotional release
- Red Axes integration for weird dark industrial twist
- Ghost entry: Lauren Garnier - The Man With The Red Face (extended loop)
- Exit strategy: Mills Late Night dubby dark fade
