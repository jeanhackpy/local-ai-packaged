## Résumé

Blueprint for streaming data to RAG. NVIDIA Holoscan for signal processing, Riva ASR (Parakeet CTC), Llama NeMo Retriever 3.2 for embeddings. Sub-5-second latency speech-to-searchable-insight. Continuous document ingestion.

## Points clés

- NVIDIA Holoscan handles signal processing pipeline
- Riva ASR (Parakeet CTC) for speech recognition, Llama NeMo Retriever 3.2 for embeddings
- Sub-5-second latency with continuous document ingestion for real-time indexing
