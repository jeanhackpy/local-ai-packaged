## Résumé

PRD pour l'intégration Sansiri, premier développeur immobilier thaïlandais. Révèle la structure côté développeur : projet → bâtiment → unité. Types condos + BTR (Build-to-Rent) + maisons individuelles. Segment BTR émergeant en Thaïlande.

## Points clés

- Hiérarchie données développeur : project > building > unit
- Types de propriétés : condo, Build-to-Rent (BTR), maisons individuelles
- Segment BTR (Build-to-Rent) comme segment émergent en Thaïlande
- Intégration via API pour availability et pricing en temps réel
