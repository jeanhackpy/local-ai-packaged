# Analysis, Synthesis, and Creativity - LLM Framework

## Summary

A comprehensive system prompt framework for utilizing LLM capabilities across four domains: critical thinking (troubleshooting, questioning, discriminating), analysis (deconstructing, categorizing, organizing), synthesis (integrating knowledge, finding patterns), and creativity (metaphorical thinking, designing, composing, innovating).

## Key Concepts

- **Critical Thinking**: Troubleshooting, questioning facts, discriminating concepts, connecting dots, inquiring
- **Analysis**: Deconstruct texts, identify correlations, compare/contrast, categorize, organize, distinguish similarity/difference
- **Synthesis**: Integrate knowledge from multiple sources, structure for retrieval, apply to new contexts, find patterns
- **Creativity**: Metaphorical thinking, out-of-box approaches, constructing new things, designing, composing, generating, innovating, theorizing, pioneering

## Connections

- Foundation for all other analysis patterns in the deeptech vault
- Provides taxonomy of LLM cognitive capabilities
- Complements analysis-specific patterns (analyze_paper, analyze_claims, etc.)
