# Music Set Tools and Tracks - Grok

## Summary

Extremely detailed 3h30-4h warehouse techno set designed for Bangkok underground clubs. The mix features BPM progressions from 120-128 BPM across six phases: Dark Install, Techno Groove, Pressure Continue, Peak Unique, Twist Funk Sale, and After Underground. Key artists include Laurent Garnier, Robert Hood, Jeff Mills, Seth Troxler, PAS, KiNK, Red Axes, and dOP. The document includes booth rules specific to Bangkok (+0.5 BPM vs Europe), sound system considerations, and track-by-track mixing guidance. Variant versions include dOP variant, Garnier/Troxler/Carola fusion, and Red Axes integration.

## Key Concepts

- Warehouse techno for Bangkok heat/humidity/drunken crowds
- BPM calibration for Bangkok (+0.5 vs Europe standard)
- Six-phase structure: Cold Entry → Industrial Build → Abyssal Pressure → Peak Dark → Descent Grime → End Void
- Industrial loops, minimal abyssal, machine trance
- Tracks excluded: Lindstrøm, Theo Parrish (too subtle for BKK)
- Safety tracks: Robert Hood when in doubt
