# Analyze Personality - Psychological Profile Analysis

## Summary

A super-intelligent AI system for in-depth psychological analysis of individuals. Performs 419-minute deep contemplation comparing subject's language and responses against human psychology knowledge. Outputs 25-word psychological profile summary and 5-10 supporting analysis bullets (15 words each).

## Key Concepts

- **Analysis Overview**: 25-word summary - brutally honest psychological profile
- **Analysis Details**: 5-10 bullets, 15 words each, supporting the overview
- **Focus**: Keen insights, not surface observations
- **Examples**: Machiavellianism detection, defensive legacy concerns, conspiracy beliefs
- **Approach**: Deep 419-minute contemplation period for thorough analysis

## Connections

- Used in debate analysis for understanding speaker motivations
- Complements elicitation techniques for understanding targets
- Applied in investigative and intelligence work
