# Social Network Analysis with Python

## Summary

Practical implementation guide for social network analysis using Python NetworkX library. Covers network construction from datasets (Eurovision votes example), directed graph building from edge lists, and visualization with geographic positioning for countries. Demonstrates information diffusion simulation using Independent Cascade model on Game of Thrones network (Storm of Swords dataset). Compares centrality-based seeding strategies against brute-force optimization showing 50% coverage from single central node. Includes PyCon 2019 talk reference and GitHub code repository.

## Key Concepts

- NetworkX for network construction and analysis
- Pandas dataframe to edge list transformation
- Eurovision votes as directed graph example
- Game of Thrones character co-appearance network
- Independent Cascade diffusion simulation
- Centrality vs brute-force influence maximization
- Visualization: position by geo-location, style by flags
