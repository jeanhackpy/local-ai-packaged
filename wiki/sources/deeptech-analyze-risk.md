# Analyze Risk - Third-Party Vendor Risk Assessment

## Summary

A comprehensive vendor risk assessment framework for analyzing third-party compliance with security and privacy standards. Assigns Low/Medium/High risk scores based on documents (UW IT Security Terms Rider, DPA, vendor website) and suggests necessary security controls for users/implementers.

## Key Concepts

- **Risk Score**: Low, Medium, or High rating
- **Document analysis**: Security Terms Rider, DPA, vendor website evaluation
- **Compliance check**: State laws, federal laws, university policies
- **Security controls**: Password management, incident response, data backup, patching
- **Output**: Detailed reasoning document for risk score
- **Use case**: Procurement requests, vendor evaluation
