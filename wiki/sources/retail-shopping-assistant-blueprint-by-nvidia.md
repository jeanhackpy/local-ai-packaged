# Retail Shopping Assistant Blueprint by NVIDIA

## Summary

AI Blueprint for intelligent shopping assistants delivering personalized product recommendations, handling complex/long-tail queries, and driving higher conversion rates. Features multimodal, multi-query agentic RAG with text and image capabilities.

## Key Concepts

- **Multimodal RAG Pipeline**: Text + image-to-image similarity search with NVClip
- **Llama 3.1 70B NIM**: Natural, humanlike conversational interactions
- **NeMo Guardrails**: Topic control and content safety for brand-safe conversations
- **NVIDIA Retrieval QA E5 v5**: High-accuracy information retrieval with data privacy
- **cuVS-accelerated Milvus**: GPU-accelerated vector database for fast similarity search
- **LangChain Integration**: Flexible orchestration framework
- **Cross-sell/Upsell**: Intelligent complementary product suggestions
