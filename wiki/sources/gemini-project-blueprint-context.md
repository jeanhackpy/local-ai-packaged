# Gemini Context - Obsidian Vault for Project REcall

## Summary

Overview document describing the Gemini-obsidian directory as the shared knowledge base for Project REcall collaboration. The vault manages strategy, architecture, and daily interactions for developing the REcall real estate intelligence platform. The document serves as a quick-reference guide to the vault's structure and the project's high-level vision.

## Key Concepts

- **REcall OS**: "Palantir for Real Estate" - AI-powered operating system for Southeast Asian real estate
- **Three Core Brands**: REcall Agency (tech/cybersecurity), Reflexion.asia (B2C/B2B marketplace), Patrimonasia.com (B2B European investor bridge)
- **Tech Stack**: Next.js, TailwindCSS, Supabase, Qdrant, Neo4j, Redis, n8n, LangGraph/Flowise, OpenRouter, Ollama
- **3D Layer**: Unreal Engine 5 + Cesium for immersive property experiences
- **Vault Structure**: Daily logs in `logs/daily_interactions/`, project notes in `projects/`, data in `data/`
