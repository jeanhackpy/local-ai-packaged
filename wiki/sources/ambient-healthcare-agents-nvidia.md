# Ambient Healthcare Agents

## Summary

Dual agent architecture for healthcare: Provider Voice Agent (clinical documentation/SOAP notes) and Patient Voice Agent (intake, scheduling, information queries). Ambient AI for clinical workflows.

## Key Concepts

- **Dual agent system**: Provider + Patient agents
- **Provider Agent**: Clinical documentation from patient conversations
- **Patient Agent**: Intake, scheduling, information queries
- **Ambient AI**: Background clinical documentation capture
- **SOAP notes**: Structured clinical note generation
